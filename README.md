# 🤖 Chatbot Telegram com IA

Bot do Telegram com integração de Inteligência Artificial usando OpenRouter API.  
Desenvolvido em Python para fornecer respostas automáticas e inteligentes em tempo real.

---

## 🚀 Funcionalidades

- 🤖 Respostas automáticas com IA
- 💬 Integração com Telegram
- 🌐 Uso da API OpenRouter
- ⚡ Simples e fácil de configurar
- 🇧🇷 Respostas em português

---

## 🛠️ Tecnologias utilizadas

- Python
- pyTelegramBotAPI
- Requests
- OpenRouter API
- python-dotenv
